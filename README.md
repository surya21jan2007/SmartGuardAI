# SmartGuard AI – Camera Theft Detection System

## Objective
Detect camera theft attempts in real-time and send alerts with evidence.

## Technologies
- Python
- OpenCV
- NumPy
- YOLOv8 (Ultralytics)
- Telegram Bot API

## Features
- Motion Detection
- Human Detection
- Brightness Analysis
- Camera Shake Detection
- Theft Detection Logic
- Evidence Saving
- Telegram Alert Integration

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run notebook: `jupyter notebook smartguard_ai.ipynb`
3. Or run script: `python smartguard_ai.py`
